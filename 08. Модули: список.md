Чтобы посмотреть **все доступные модули Ansible**, используйте:

```bash
ansible-doc -l
```

Эта команда покажет полный список всех модулей, установленных в системе.

**Другие полезные варианты:**

```bash
# Список модулей с фильтром (например, все модули для работы с файлами)
ansible-doc -l | grep file

# Подробная информация о конкретном модуле
ansible-doc copy

# Краткая информация о модуле
ansible-doc -s copy

# Список всех плагинов (не только модулей, но и инвентори, lookup, filter и т.д.)
ansible-doc -t module -l
```

**Что касается ошибок `ansible-galaxy list`** — они возникают потому, что у вас не созданы стандартные директории для ролей. Это нормально, если вы их не использовали. Чтобы их создать:

```bash
mkdir -p ~/.ansible/roles
```

Или игнорируйте эти предупреждения, если роли вам не нужны.
